# Generador de Cifrado y Análisis de Frecuencia en Java
Este ejercicio incluye una serie de Clases en Java que permiten descifrar en forma de fuerza bruta, textos utilizando distintos algoritmos de cifrado y descifrado clásico, como Caesar, Afín y Vigenere. También existe una clase que realiza un análisis de frecuencia de aparición de letras en un texto cifrado en comparación con la distribución teórica del español. Esto con el propósito de proponer las mejores 5 llaves más probables según el análisis de frecuencia.
## Funcionalidades
1. Cifrados
- Descrifrado César (Fuerza bruta)
- Descifrado Afin (Fuerza bruta)
- Descifrado Vigenère (Fuerza bruta)

2. Análisis de Frecuencia
- Análisis de probabilidad para el alfabeto en español para proponer las llaves más prometedoras para descrifrar el código.

# Cómo Ejecutar el Código
1. Descargar el proyecto.
2. Compilar cada archivo por separado (CaesarBruteforce.java, AfinBruteforce.java & VigenereBruteforce.java).
3. Ejecutar el cada archivo compilado, dependiendo del tipo de descifrado a fuerza bruta que se quiera realizar.
